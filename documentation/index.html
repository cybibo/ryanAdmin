RYANADMIN Documentation

Designed by: RyanDesigns
Template version: 1.6
Date Created: November 15, 2010
Last Update: Feb 24, 2011

RYANADMIN is a complete control panel template. It includes all the necessary styling for a content management system such as graphs, media, pricing tables and progress bar.

It's ideal for content management systems, website backends or as a stand alone skin for a web application.

tables.css

Tables.css contains all the necessary styling for the tables. (More Examples on tables.html)

Here are some example usages for the table.

Automatically enable pagination on a table
To automatically paginate a table, add the paginate class 

<table class="paginate">
</table>

Manually enable pagination on a table

 <table class="mytableclass">
</table>

Add the following javascript

 $(document).ready(function(){
	$("table.mytableclass").paginate({rows: 10, buttonClass: 'button-blue'});
});

rows: The number of rows to show per page
buttonClass: button-gray, button-blue, button-green, button-orange

Automatically enable sorting on a table
To automatically allow column sorting on a table, add the sortable class 

<table class="sortable">
</table>

Manually enable sorting on a table

<table class="mytableclass">
</table>

Add the following javascript

$(document).ready(function(){
	$("table.mytableclass").tablesort();
});

global.js

This include all the necessary scripts needed for all the pages.


html5.js

This supports all the newly html tags used in this template. It is a way for browsers to acknowledge the new elements such as <article>,<section>, etc.


ie.js

It is a JavaScript library to make Microsoft Internet Explorer behave like a standards-compliant browser. It fixes many HTML and CSS issues such as mousedown,mouseup and mouseout.


jquery.animate-shadow-min.js

With this jQuery plugin, you can extend the animate function to support the CSS box shadow-property. You can animate the color, the x- and y-offset, the blur-radius and spread-radius. Mark Carver contributed code to support the alpha channel.


jquery.tables.js

Tablesorter is a jQuery plugin for turning a standard HTML table with THEAD and TBODY tags into a sortable table without page refreshes. tablesorter can successfully parse and sort many types of data including linked data in a cell. It has many useful features including:

  • Multi-column sorting
  • Parsers for sorting text, URIs, integers, currency, floats, IP addresses, dates (ISO, long and short formats), time. Add your own easily
  • Support secondary "hidden" sorting (e.g., maintain alphabetical sort when sorting on other criteria)
  • Extensibility via widget system
  • Cross-browser: IE 6.0+, FF 2+, Safari 2.0+, Opera 9.0+
  • Small code size

jquery.tools.min.js

jQuery Tools is a collection of the most important user interface components for the web. These are tabs, accordions, tooltips, overlays, exposing effects and scrollables. It is minimized and/or packed to reduce the size of the files.


pricing.js

This is a Javascript used in pricing table. More example on pricing.html


selectivizr.js

selectivizr is a JavaScript utility that emulates CSS3 pseudo-classes and attribute selectors in Internet Explorer 6-8.


jquery.easing.1.3.js

A JQuery extension that allows you to set custom easing types for standard animations. To learn more about jquery.easing, click here


jquery.quicksand.js

Quicksand is a very functional jQuery plugin for reordering and/or filtering items with a nice shuffling animation.

It simply replaces a collection of items with another & only requires the sets to be provided which can be done in multiple ways:

  • using plain HTML, like an unordered list
  • downloading data by an Ajax call
  • transforming HTML items by JavaScript

To use quicksand, simply add this jquery script:

$("#source").quicksand($sorted_data, $preferences); </div>
where in:
source - list of the objects
sorted-data - new list of the objects after sorted and filtered
preferences - additional animation parameters such as easing of the animation and duration of the animation,

For more demo on using quicksand, click here.
